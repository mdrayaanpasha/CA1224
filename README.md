# CA1224 Repository

Digital Archieve of CA1224… SJU’s first class on programming: ** Programming with C ** repository!
I made it during during my lectures at St. Joseph's University (SJU). 
It serves as a personal archive and a learning resource for fellow students.

## 📂 Project Structure

The project has been reorganized into logical categories for better navigability:

- **`src/basics/`**: Fundamental C programs including math formulas, simple logic, and basic I/O.
- **`src/arrays/`**: 1D array operations, including searching, maximum finding, and statistical calculations (Average, SD).
- **`src/matrices/`**: 2D array and matrix operations like addition, finding the maximum, and trace calculation.
- **`src/series/`**: Programs involving loops and mathematical series (Factorial, Taylor series, Sum series).
- **`src/search_sort/`**: Implementation of common algorithms like Bubble Sort and Binary Search.
- **`src/file_io/`**: Experiments with file handling in C (copying, reading, and writing files).
- **`src/lab/`**: More complex programs, including student management and salary calculations.
- **`scripts/`**: Supporting scripts (e.g., Python scripts used for testing or matching).

## 🌐 Web Interface

You can navigate through the lecture-wise source code using the included [link](https://mdrayaanpasha.github.io/CA1224/). It provides a clean, categorized view of all the exercises linked to their respective dates and lecture numbers.

## 🚀 Getting Started

To run any of the C programs locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/mdrayaanpasha/sju-c-basics.git
   ```
2. Navigate to the desired file in `src/`.
3. Compile using a C compiler (like `gcc`):
   ```bash
   gcc filename.c -o filename
   ```
4. Execute the program:
   ```bash
   ./filename
   ```
